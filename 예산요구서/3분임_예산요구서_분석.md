## Ⅳ. 제출 전 보완 리스트 (우선순위)

```
[최우선 — 실점 직결]
① 장애요인·극복방안 표 신설 (13세부사업별 1:1)   ← E2 1점→5점  ✅ 완료(보완_장애요인_극복방안.md)
② 참고1~3 완성: 이해관계자 3자 인터뷰 + 유사중복 검토표   ← B4·C3  ⬜ 진행 필요
③ 오타 수정('생활레육' 등) + 목차 정리 + 서식 통일   ← F2  ⬜

[우위 강화 — 격차 확대]
④ 지원필요성 통계를 '장애인 X% vs 비장애인 Y%' 격차형으로 재서술   ← B2
⑤ 해외사례에 '국내 적용방안' 1줄씩 추가 (제도명→우리 세부사업 연결)   ← E1
⑥ 중앙정부 책무 논거 강화: 시장 미공급·헌법적 권리·지역격차 명문화   ← B3

[검토]
⑦ 세부사업 13개 통폐합 여지 검토 (예: 인식개선 광고+공모전 통합)   ← A1
⑧ 국고보조율 70% 근거 항목별 정교화 (체육≠사회복지 반론 대비)
```

**핵심 판단**: 산출근거(C축)는 이미 최상급 — 여기서 더 파지 말 것. 남은 승부처는 **②③으로 형식 실점을 지우는 것**. 우리 주제의 공공성·즉시성(채점자 즉시 공감)이 핵심 자산.
