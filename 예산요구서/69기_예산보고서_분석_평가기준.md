## Ⅰ. 공식 채점 프레임워크 (3W)

공식 채점표는 예산요구서를 **3개 질문 = 3개 평가축**으로 본다. 모든 조언·검증은 이 축에 매핑한다.

| 축 | 핵심 질문 | 평가 결론 | 세부 판단요소(공식 문구) |
|----|-----------|-----------|--------------------------|
| **What** | 무엇을 어떻게 지원? | **사업내용의 명확성** | 회계구분, 지방비 분담(전액/일부 보조), 지원방식(보조·융자), 추진주체(직접·위탁), 사업목적↔세부사업 연관성, 구체적 실행계획 |
| **Why** | 왜 중앙정부가 지원? | **지원필요성의 설득력** | 사업목적·추진근거, 중앙정부 책무, 재정지출 필요성, 사업타당성 객관적 검증, 시급성, 비용대비 효과성 |
| **How much** | 대상·범위·금액은? | **산출근거의 타당성** | 물량·단가의 근거, 잠재 부대비용, 여건 성숙성(사전절차), 유사·중복성(타 사업·조세지출), 집행가능성 |

**작성 형식 요건(감점 방지선)** — 채점표 별도 명시:
- 중2 수준으로 쉽게, 전문용어·암호 풀어쓰기
- 표에 **합계·단위 누락 금지**, 표마다 설명 문장 동반
- 모든 문장은 **뒷받침 통계수치와 함께** 제시
- **숫자 오타** 점검, 볼드·소목차 활용, 글씨체·자간·숫자정렬 통일
- ⚠️ **주제 선정 금기**: 예타 대상(대규모 건축·토목·정보화), R&D 등 고도 전문지식 필요 사업 지양 → 산출근거로 승부 어려움

## Ⅱ. 69기 우수 3편의 공통 골격 (표준 목차)

세 편이 **동일한 4부 구조**를 따랐다. 이 골격에서 벗어나면 형식 감점.

```
[표지] 열린재정 사업기본정보표 (①부처 ②회계/계정 ③12대분야 ④정책과제 '모두 병기')
       + 예산 총괄표 (백만원, '23결산~'25요구, 세부사업별 금액·증감·%)

1. 사업개요
   - 사업개요표 (사업내용/기간/총사업비[국고 분리]/규모/지원조건/수행주체/기대효과)
   (1) 지원근거 및 추진경위
       □ 지원근거  : 국정과제 연계 → 대통령·총리 말씀 인용 → 근거법령 표(세부사업별 조문 매핑) → 중장기계획
       □ 추진경위  : 배경 → 제도 경과(연혁 표: 기관·정책·내용·시기)

2. '25년 요구내용
   (1) 총괄표 / 요구방향
   (2) 요구방향 및 지원필요성
       □ 현황 및 문제점 (세부사업별, 통계·인터뷰·기사로 논증)
       □ 추진방향 (문제 → 대안 로직 다이어그램)
   (3) 세부 요구내용
       - 사업개요 (세부사업별 표: 대상/정책내용/소요금액/추진체계/모니터링/기대효과)
       - 세부 내용 (세부사업별 산출근거: 단가×수량×국고보조율, 근거 각주)

3. 중기재정 소요전망 ('25~'29)
   (1) 사업운영기본방향 : 연도별 성과목표(5개년), 중기 추진방향
   (2) 요구내용 : 중기 규모 추이 + 연평균 증가율, 연도별 산출근거

4. 고려사항
   - 외부기관 지적사항 / 전체·세부 추진절차 / 외국·민간 사례 / 예상 장애요인 및 극복방안

[참고·붙임] 산출근거 상세 (단가 산정기준, 유사·중복 검토, 인터뷰 전문, 시범사업안)
```

**규모감 참고**: 1분임 붙임 10종+참고 3종(약 80p), 12분임 붙임 4종+참고 1종. → 붙임의 두께 = 산출근거 타당성 점수.

## Ⅲ. 채점항목별 우수 패턴 (3편 근거 추출)

### 1. What — 사업내용의 명확성

| 우수패턴 | 3편 근거 예시 |
|----------|---------------|
| **사업목적 1개 → 세부사업 4~5개로 MECE 분해** | 1분임: 정책기반·출산·돌봄·교육·인식 5개 / 12분임: 합숙재활·플랫폼·눈맞춤·인식 4개. **12개 이상은 비MECE 감점** |
| **지원조건을 세부사업별로 차등 명시** | 10분임: ①-1직접(국비100%) / ①-2지자체보조(50:50) / ② 보조… 세부사업마다 회계·보조율 분리 표기 |
| **추진주체 = 주관기관+참여기관 분리** | 1분임: (주관)복지부 (참여)교육부·지자체·KODDI·보사연 등. 위탁대상 기관명 실명 |
| **세부사업 개요표 6요소 고정** | 대상 / 정책내용 / 소요금액 / 추진체계 / **모니터링** / 기대효과 — 모니터링 칸이 효율성 점수 직결 |
| **회계/계정 실제 특별회계까지 지정** | 10분임: '농어촌구조개선특별회계' / 12분임·1분임: 일반회계. 재원 성격 근거 있음 |

### 2. Why — 지원필요성의 설득력

| 우수패턴 | 3편 근거 예시 |
|----------|---------------|
| **국정과제 번호 + 추진방향 직접 인용** | 1분임: 국정 44·46·47번 / 12분임: 63-7 '전주기 마약 안전망' / 10분임: 72번 '식량주권'. 번호·문구 원문 |
| **최고위층 말씀 인용으로 시급성 근거화** | 1분임: 국정감사·UN권리위 / 12분임: 대통령 주례회동·법무부장관 발언 / 10분임: 국무총리 주례회동 |
| **통계 = 우리집단 vs 비교군 격차** | 산모 상급의료기관 이용률 25.7% vs 비장애 15.5% / 장애인 취업률 37.2% vs 전체 63.3% / 산후조리원 이용률 22% vs 81.2% |

| **당사자·전문가 인터뷰 → 특정 세부사업 직결** | "청각장애부모는 아이 울음 못 들어" (뇌병변 D씨) → **응급대응 패키지**로 1:1 연결 |
| **구조적 문제임을 입증** (일시적 사건 아님) | 지자체 조례 부재 3곳, 지역 간 예산 편차(서울 17.7억 vs 부산 0.3억) → 전국 단위 개입 필요 |
| **중앙정부 책무 = 지자체 편차 해소 논리** | 지자체 자율에 맡기면 편차 심화 → 국가 표준·재정지원 필요 (3편 공통 논법) |

|---|----------|----------------------|:----:|:-----:|:------:|
| **A. What (명확성)** ||||||
| A1 | 세부사업 MECE 분해 | 4~8개, 중복·누락 없음 | 5 | | |

## Ⅴ. 질문 공방 설계 원칙 (2문 공격 / 방어)

> ※ 실제 질문은 3분임·14분임 보고서 확보 후 확정. 여기서는 표적 선정 원칙만.

**우리가 던질 2문 (상대 최약점 저격)** — 우선순위:
1. **C2 산출근거 출처 공백** — "○○ 단가의 근거가 무엇인가? 공식 표준(학술용역단가/SW대가가이드) 준용인가, 임의 추정인가?" → 답 못하면 산출근거 타당성 붕괴
2. **C3 유사·중복** — "기존 △△사업/급여와 중복 지원 아닌가? 배타적 설계 근거는?" → 재정 낭비 프레임
3. **B2/B3 비교군·구조성** — "제시 통계가 비교군 대비 격차인가? 일시적 사건 아닌 구조적 문제라는 근거는?"
4. **A1 MECE / 포괄적 대안** — "세부사업이 모든 문제를 다 담으려다 초점을 잃은 것 아닌가?"

→ **원칙**: 질문은 '반박 불가능한 사실 확인형'으로. 답변 시 산출식·출처를 못 대면 실점하는 구조로 설계. 2발 모두 How much(C축)에 집중하면 위력 최대(승부처이기 때문).

**우리가 받을 예상 질문 (방어 준비)**:
- 우리 루브릭 **C·D축 최저점 항목** = 상대의 1순위 표적 → 해당 산출식·출처·검산을 사전 암기
- 방어 답변 3종 세트: ①단가 출처 1문장 ②수량 근거 1문장 ③국고보조율 논리 1문장
