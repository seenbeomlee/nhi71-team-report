## 3. 상위 팀 상세 분석

### 69기 1분임 (1등, 장애부모 양육지원, 총점 28.33)

**정책보고서**
- 목차: 임신→출산→돌봄→교육→소통 생애주기로 11개 문제영역 MECE 구성
- 실증: 장애산모 428명 마이크로데이터 직접 재구성, UN 권고, 영미·호주·독일 사례
- 대안: 중앙장애가정지원센터 신설, 통합정보플랫폼 '함께키움', 장애친화 산부인과 50개, 스마트응급패키지
- 예상장애물: 의료기관 참여 거부 → 장애친화 산부인과 인센티브로 대응

**법제보고서**
- 「장애부모 양육지원법」 신규 제정 46조 + 부칙
- 돌봄지원사 국가자격제도 신설, 이의신청·청문·과태료까지 포함
- 법률-시행령 2단계 완결

**예산보고서**
- 총 33,345백만원 (5개 군)
- SW기능점수(605,784원/FP)×보정계수(규모·연계복잡성·성능·운영환경·보안성)로 단가 산출
- 장애인복지법 시행령 국고보조율 명시
- 5년 중기 연평균 +10.62%

**법제보고서**
- 산업안전보건법 + 시행령 + 시행규칙 + 「산업안전보건기준에 관한 규칙」 + 안전보건교육규정

**예산보고서**

**예산보고서**: 총 18,616백만원. 이송 구급차 산출("광역 17개소×1대=17대")이 가정 수준.

## 5. 2년치 공통 패턴 — 결론

### 상위팀 7가지 공통 패턴

1. **분류 원칙 1개 선언** — 유형별(현장/이동), 단계별(대비→발생→적응), 인과적(문제→원인→대안) 중 택1. 보고서 첫 페이지에 선언, 전체 구조에 일관 적용. 문제영역 5~8개 적정.

2. **통계 = 비교군 대비 격차** — 절댓값 단독이 아니라 "우리 집단 X% vs 비교군 Y%"로 시급성 증명.

3. **이해관계자 3자 구성** — 정책담당자·전문가·당사자(수혜자). 각 인터뷰 의견을 특정 대안 선택 근거와 1:1 연결.

4. **법제 고난이도 전략 2가지 중 택1**:
   - 신법 제정: 36조문 이상 완결 체계 (69기 1분임 46조, 70기 4팀 36조)
   - 핵심법 다층 개정: 법률-시행령-시행규칙-고시 전 계층 동시 정비 (69기 12분임 7개법령, 70기 7팀 5개법령)

5. **예산 단가 = 공신력 있는 가이드라인명 명시**:
   - 인건비: 「행안부 학술연구용역 인건비 기준단가」
   - SW개발비: 「SW사업 대가산정 가이드 FP방식」
   - 주택비: LH 매입임대주택 전세 중위값
   - 중기 물가·임금 상승률: 한국은행 경제전망자료

6. **해외사례 = 프로그램명+법령명+조항+국내 적용 연결** — "A국에서도 유사 정책 시행 중" 수준 불인정.

7. **야심적 조직개편 제안 시 실현가능성 증명** — 선례·이해관계자 지지 근거·단계적 추진계획 필수.

### 하위팀 4가지 실패 패턴

1. **표 의존** — 텍스트 논증 부재, 채점자가 논리 흐름 파악 불가
2. **예산 재조정 이력** — 초기 설계 불안정 노출 (70기 12팀 37%, 17팀 대상자 수 재산출)
3. **포괄적 대안** — 모든 문제를 한꺼번에 해결, 각 대안 논증 깊이 얕음
4. **주제 이탈** — 핵심과 거리 있는 대안 포함 (9팀: 6차산업 이탈, 15분임: 청년당뇨에 홍보 과다)

## 6. 71기 실전 체크리스트

### 정책보고서 작성 전
```
□ 분류 원칙 1개 결정 (유형별/단계별/인과적 중 택1)
□ 문제영역 5~8개로 제한 (12개 이상은 비MECE)
□ 핵심 통계마다 "우리 대상 X% vs 비교군 Y%" 형식으로 수치 확보
□ 이해관계자 인터뷰 대상자 3명 사전 확정 (정책담당자·전문가·당사자)
□ 각 인터뷰 결과 → 어느 대안 선택에 반영할지 사전 설계
□ 대안 비교표 작성 계획 ("왜 이 대안을 선택하지 않았는가")
```

### 정책보고서 작성 중
```
□ 문제영역 간 중복·누락 없는지 확인 (MECE 검증)
□ 해외사례: 프로그램명+법령명+조항+국내 적용 방안까지 서술
□ 예상 장애물 항목마다 구체적 극복방안 병기
□ 포괄적 대안 없는지 확인 — 모든 문제 해결하려 하지 않기
□ 이해관계자 인터뷰 의견 → 특정 대안과 직결 명시
```

### 법제보고서 체크

```
□ 전략 확정: 신법 제정(36조+) vs 핵심법 다층 개정(법률-시행령-시행규칙-고시)
□ 신구조문대비표 포함
□ 헌법 근거 또는 국제기준 명시
□ 핵심 조항이 "부령에서 정한다"로 끝나지 않도록 (직접 명령 방식)
□ 벌칙·과태료·부칙(시행일·경과조치) 완비
□ 법령 간 위계적 연결 확인 (법률→시행령→규칙→고시)
```
