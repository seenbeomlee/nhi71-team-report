## 71기 실전 체크리스트

### 정책보고서 작성 전
```
□ 분류 원칙 1개 결정 (유형별/단계별/인과적 중 택1)
□ 문제영역 5~8개로 제한 (12개 이상은 비MECE)
□ 핵심 통계마다 "우리 대상 X% vs 비교군 Y%" 형식으로 수치 확보
□ 이해관계자 인터뷰 대상자 3명 사전 확정 (정책담당자·전문가·당사자)
□ 각 인터뷰 결과 → 어느 대안 선택에 반영할지 사전 설계
□ 대안 비교표 작성 계획 ("왜 이 대안을 선택하지 않았는가")
```

### 정책보고서 작성 중
```
□ 문제영역 간 중복·누락 없는지 확인 (MECE 검증)
□ 해외사례: 프로그램명+법령명+조항+국내 적용 방안까지 서술
□ 예상 장애물 항목마다 구체적 극복방안 병기
□ 포괄적 대안 없는지 확인 — 모든 문제 해결하려 하지 않기
□ 이해관계자 인터뷰 의견 → 특정 대안과 직결 명시
```

### 법제보고서 체크
```
□ 전략 확정: 신법 제정(36조+) vs 핵심법 다층 개정(법률-시행령-시행규칙-고시)
□ 신구조문대비표 포함
□ 헌법 근거 또는 국제기준 명시
□ 핵심 조항이 "부령에서 정한다"로 끝나지 않도록 (직접 명령 방식)
□ 벌칙·과태료·부칙(시행일·경과조치) 완비
□ 법령 간 위계적 연결 확인 (법률→시행령→규칙→고시)
```
