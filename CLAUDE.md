## 【검토 중】 3Go 목차 개편안 (2026.07.05 작성, 반영 미확정)

> **⚠️ 아직 확정된 구조가 아니다.** 확정 전까지 모든 파트 AI는 위 "3대 쟁점 구조(유형별)"를 기준으로 작업하고,
> 이 개편안을 근거로 `자료조사/` 하위 파일이나 파트 CLAUDE.md를 수정하지 않는다.

- **무엇인가**: 보고서 목차의 분류 원칙을 기존 **유형별**(인프라/프로그램/관리)에서 **참여 단계별**(준비→참여→지속)로 바꾸는 안
- **어디에 있나**: `restructure/3go-outline` 브랜치의 **`3Go_목차개편안.md`** (저장소 루트, main에는 없음)
  - 전체 내용을 보려면 `git show restructure/3go-outline:3Go_목차개편안.md` 실행
- **네모(ㅁ) 3개**: **준비하Go**(참여 진입) / **즐기Go**(참여 경험) / **이어가Go**(참여 지속)
- **동그라미(ㅇ) 재배열**: 기존 중목차들을 아래처럼 새 동그라미로 묶는다. 괄호 안이 동그라미 키워드.

| 네모 | 문제점 ㅇ(키워드) → 기존 중목차 | 개선방안 ㅇ(키워드 제안) |
|------|------------------------------|----------------------|
| 준비하Go | (시설부족)=시설부족+야외환경+장비부족 / (교통미비)=입지+교통 / (홍보부족)=홍보 | (시설확충) / (접근제고) / (홍보강화) |
| 즐기Go | (프로그램)=콘텐츠+환경 / (인력)=양적한계+질적한계 / (분리)=분절성+동기부족 | (수요맞춤) / (인력확대) / (어울림) |
| 이어가Go | (플랫폼)=플랫폼+바우처 / (운영·관리)=운영비+환류체계 / (노후시설)=실내환경 | (원스톱) / (운영개선) / (시설개선) |

- **키워드 상태**: 문제점 키워드는 팀 예시 확정값, 개선방안 키워드는 AI 제안값(미확정)
- **주의 (미해결 쟁점)**: ① 야외환경(준비하Go)과 실내환경(이어가Go)이 다른 네모로 갈라짐 — "신규 공급 vs 기존 유지관리" 구분 논리 필요 ② (노후시설)을 이어가Go에 두는 근거가 약함 ③ 새 구조는 파트 담당과 네모가 1:1이 아니므로, 확정되어도 `자료조사/` 폴더 구조는 유지하고 보고서 조립 단계에서만 재배열한다
- **확정 시 할 일**: 팀장이 이 섹션을 "확정"으로 갱신 + `restructure/3go-outline`을 main에 머지

   - 물가·임금: 한국은행 경제전망자료
6. **해외사례 깊이** — 프로그램명+법령명+조항+국내 적용 연결 (국가명 수준 불인정)
7. **조직개편 실현가능성** — 선례+이해관계자 지지+단계적 추진계획 필수

### 하위팀 실패 패턴
- 표 의존(텍스트 논증 부재), 예산 재조정 이력(30~37% 총액 변경), 포괄적 대안, 주제 이탈
- 이해관계자 인터뷰 의견이 특정 대안과 직결되지 않음 (인터뷰 나열에 그침)

## 71기 실전 체크리스트

### 정책보고서 작성 전
```
□ 분류 원칙 1개 결정 (유형별/단계별/인과적 중 택1)
□ 문제영역 5~8개로 제한 (12개 이상은 비MECE)
□ 핵심 통계마다 "우리 대상 X% vs 비교군 Y%" 형식으로 수치 확보
□ 이해관계자 인터뷰 대상자 3명 사전 확정 (정책담당자·전문가·당사자)
□ 각 인터뷰 결과 → 어느 대안 선택에 반영할지 사전 설계
□ 대안 비교표 작성 계획 ("왜 이 대안을 선택하지 않았는가")
```

### 정책보고서 작성 중
```
□ 문제영역 간 중복·누락 없는지 확인 (MECE 검증)
□ 해외사례: 프로그램명+법령명+조항+국내 적용 방안까지 서술
□ 예상 장애물 항목마다 구체적 극복방안 병기
□ 포괄적 대안 없는지 확인 — 모든 문제 해결하려 하지 않기
□ 이해관계자 인터뷰 의견 → 특정 대안과 직결 명시
```

### 법제보고서 체크
```
□ 전략 확정: 신법 제정(36조+) vs 핵심법 다층 개정(법률-시행령-시행규칙-고시)
□ 신구조문대비표 포함
□ 헌법 근거 또는 국제기준 명시
□ 핵심 조항이 "부령에서 정한다"로 끝나지 않도록 (직접 명령 방식)
□ 벌칙·과태료·부칙(시행일·경과조치) 완비
□ 법령 간 위계적 연결 확인 (법률→시행령→규칙→고시)
```

### 예산보고서 체크
```
□ 세부 항목마다 단가×수량×횟수 3요소 각각 출처 명시
□ 인건비: 「행안부 학술연구용역 인건비 기준단가」 준용
□ SW개발비: 「SW사업 대가산정 가이드 FP방식」 준용
□ 주택비: LH 매입임대주택 전세 중위값
□ 중기(5개년): 연도별 성과목표와 예산 연동 표 작성
□ 물가·임금 상승률 적용 방식 1줄 명시 (한국은행 경제전망자료)
□ 제출 전 단가 현실성 내부 검증 (재조정 이력 남기지 않기)
```
