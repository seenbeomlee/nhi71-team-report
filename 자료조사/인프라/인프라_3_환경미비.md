# 인프라 쟁점 3: 체육시설 이용 환경 미비

> 쟁점 구조: (내부환경) BF 기준 미충족 / (외부환경) 라스트마일 외부 접근로 미비  
> 개선방안: `인프라_개선방안.md` 참조

---

## 핵심 주장

> 전국 공공 실내체육시설(체육관·수영장) 2,109개소 중 실질적 장애인 접근이 가능한 BF 우수 이상 본인증 취득 시설은 286개소(13.6%)에 그치며, 그 구조적 원인은 편의증진법 시행령 별표 2의2가 신축에만 BF 의무를 부과하고 기존 시설 리모델링을 적용 제외한 법적 공백이다. 외부 접근로(라스트마일) 구간에도 BF 기준 적용 근거가 없어 시설 진입 전 단계부터 이동이 제약된다.

---

## 1. (내부환경) BF 기준 미충족 현황

### 핵심 수치

| 수치 | 값 | 출처 |
|------|-----|------|
| 공공 실내체육시설 총수 | **2,109개소** | 문체부 2024년말 |
| BF 우수이상 본인증 | **286개소** | 공단 2024.12.31 |
| **실질 BF 충족률** | **13.6%** | A×B 교차 분석 |
| 장애인 생활체육 참여율 | **24.3%** | 문체부 장애인생활체육조사 2023 |
| 비장애인 생활체육 참여율 | **62.4%** | 국민생활체육조사 2023 |
| 참여율 격차 | **38.1%p** | |

### 법적 공백 (구조적 원인)

편의증진법 시행령 별표 2의2(2015.7.24 신설) — 국가·지자체 **신축** 운동시설에만 BF 의무 부과.

| 적용 조건 | 법적 의무 |
|----------|----------|
| 국가·지자체 신축 체육관·운동장 | BF 인증 의무 (별표 2의2 제10호) |
| 국가·지자체 기존 시설 리모델링 | **의무 없음 — 법적 공백** |
| 민간 체육시설 (신축·기존 모두) | 의무 없음 (자율) |

**법적 의무가 BF 인증을 견인함을 데이터로 입증**:
- 별표2의2 신설 **전** 본인증: **6건** (2.0%) → **후**: **289건** (98.0%)
- → 기존 시설에 의무 없는 현행 구조 유지 시 13.6% 고착 불가피

### 노후화 수치 (BF 의무 미적용 규모)

| 구분 | 개소 | 비율 |
|------|------|------|
| 2015년 이전 건립 기존 시설 (BF 의무 미적용) | **1,275개소** | **60.5%** |
| 2015년 이후 신축 | 834개소 | 39.5% |
| 합계 | 2,109개소 | 100% |

### 비교군 (타 시설 대비 운동시설 BF 최하위)

| 용도 | 우수이상 본인증률 (BF 신청 시설 기준) |
|------|--------------------------------------|
| 노유자시설 (복지관 등) | **40.3%** |
| 제1종 근린생활시설 | **39.6%** |
| 업무시설 | **33.4%** |
| 전체 평균 | **34.5%** |
| **운동시설** | **30.8%** ← 최하위 |

- 운동시설 예비→본 미전환률: **68.2%** (전체 평균 64.1%보다 높음)

### 관련 법령 3개 분석

| 법령 | 조항 | BF 관련성 | 한계 |
|------|------|---------|------|
| 스포츠기본법 | 제2조 (기본이념) | 비차별 선언 | 시설 BF 기준 직접 규정 없음 |
| 장애인차별금지법 | 제25조 | 체육활동 차별금지·정당한 편의 의무 | 물리적 BF 기준 미규정 |
| 편의증진법 시행령 | 별표 2의2 제10호 | 신축 운동시설 BF 의무 | **기존 시설 리모델링 적용 제외** |

**"기존 법으로 충분하다" 반론 차단**:
- 장차법 제25조 = 행위 금지 / 편의증진법 = 시설 기준 → 규율 대상이 다름
- 스포츠기본법 제2조 = 선언적 이념 (구체적 의무·제재 없음)

### 해외 비교

| 항목 | 한국 | 일본 |
|------|------|------|
| 선언 | 스포츠기본법 제2조 | スポーツ基本法 제2조5항 (배려하며 추진) |
| 강제 | **없음** (신축만) | 배리어프리법 **제14조 제1항** (공공 체육관·수영장 2,000㎡ 이상 신축 의무) |
| 기존 시설 | **어느 법령도 BF 강제 없음** | 조례로 기준 강화 가능 (제14조 제3항) |
| 집행 구조 | 자율 인증 | 기준 적합 의무 (不適合 시 위반) |

- 영국: 장애 성인 신체 비활동 **39.5%** vs. 비장애인 **~20%** → 격차 **~20%p**  
  출처: Sport England Active Lives Survey 2023/24  
  → 한국 참여율 격차 38.1%p와 병치 가능 (측정 방식 상이, 각주 명시 필요)

## 2. (외부환경) 라스트마일 외부 접근로 미비

| 항목 | 현황 |
|------|------|
| 체육시설 외부 접근로 BF 기준 적용 법령 | **없음** |
| 이용 편의성 향상 시설 1순위 (설문) | '다니기 쉽게 만들어진 복도 및 통로' (외부 접근로 포함 해석) |
| 정량 수치 | 미확보 (향후 보완 필요) |

- 체육시설 진입 전 인도·경사로·점자블록 등 외부 접근로에 대한 BF 인증 평가 항목 없음
- 「보행안전 및 편의증진에 관한 법률」이 존재하나 체육시설 반경 외부 접근로 정비 의무 규정 미비
